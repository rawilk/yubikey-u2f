## General Requirements

-   PHP **8.1** or greater
-   Laravel **9.12** or greater
-   A [YubiKey](https://www.yubico.com/store/) security key from Yubico
-   An [API key](https://upgrade.yubico.com/getapikey/) for the Yubico servers
